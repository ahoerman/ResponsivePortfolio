Portfolio

Dog Map

In a new town and wondering where you can bring your dog? With this application, you can search any location and a map of pet friendly places is generated as well as the current weather. Using the Google Places Api and Google Maps Api the map is interactive with infowindows and a corresponding clickable list below the map.

Code Quiz

This application is a simple quiz game where users can answer questions to receive a final score which can be saved to a scoreboard using local storage.

Day Planner

An hour by hour daily planner that allows you to save items by each hour of the day. The hours in the day change based on whether they are in the past, present, or future.

Password Generator

A simple password generator. Once you have selected how long you would like the generated password to be, you can select which elements you would like it to include: uppercase letters, lowercase letters, numbers, or special characters.
